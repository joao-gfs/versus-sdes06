# Versus - Sistema de Gestão de Torneios Esportivos

Projeto desenvolvido para a matéria de **Gerência de Projetos de Sofware**.

## Cronograma
### Disponível em: [Trello](https://trello.com/b/MZ0XgMwB/versus-sdes06)

O cronograma foi criado usando como base a Estimativa de Esforço.

As extensões escolhidas usam o sistema de ***Scrum Poker*** para criar o ***Gráfico de Burndown***. 

Foi necessário converter as horas em números da sequência de fibonacci, para representar o esforço:

| Horas Estimadas | Story Points |
| :-------------: | :----------: |
| 1-4h            | **1** |
| 5-8h            | **2** |
| 9-15h           | **3** |
| 16-25h          | **5** |
| 26-40h          | **8** |
| 41-60h          | **13** |
| > 60h           | **21** (ou mais) |

Para visualizar o gráfico de burndown clique em ***"Burndown Charts for Trello"*** no cabeçalho do board do Trello.
